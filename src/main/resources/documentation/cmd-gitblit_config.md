@CMD@
================

NAME
----
@CMD@ - Query and Control Gitblit's settings.

SYNOPSIS
--------
```
ssh -p @SSH_PORT@ @CMD@
```

DESCRIPTION
-----------
Query and Control Gitblit's settings.

ACCESS
------
Caller must be an *Administrator*.

EXAMPLE
-------

```
    $ ssh -p @SSH_PORT@ @CMD@ --list
```
